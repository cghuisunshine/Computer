1. What JSON is

JSON stands for JavaScript Object Notation. It is a text format used to store and exchange structured data.

In Python, JSON is handled with the built-in json module. 

import json
JSON is usually plain text. Python converts that text into native objects like dictionaries and lists.

2. Python objects vs JSON

Python and JSON look similar, but they are not exactly the same.

Python

data = {
    "name": "Alice",
    "age": 25,
    "is_student": False,
    "skills": ["Python", "SQL"],
    "address": None
}

JSON

{
  "name": "Alice",
  "age": 25,
  "is_student": false,
  "skills": ["Python", "SQL"],
  "address": null
}

3. Convert Python object to JSON

Use json.dumps() to convert a Python object into a JSON string.

import json

data = {
    "name": "Alice",
    "age": 25,
    "skills": ["Python", "SQL"]
}

json_text = json.dumps(data)
print(json_text)

Output:

{"name": "Alice", "age": 25, "skills": ["Python", "SQL"]}

4. Pretty-print JSON

Use the indent parameter to format JSON so it is easier to read.

import json

data = {
    "name": "Alice",
    "age": 25,
    "skills": ["Python", "SQL"]
}

json_text = json.dumps(data, indent=2)
print(json_text)

Output:

{
  "name": "Alice",
  "age": 25,
  "skills": [
    "Python",
    "SQL"
  ]
}

5. Convert JSON text to Python object

Use json.loads() to parse a JSON string into a Python object.

import json

json_text = '{"name": "Alice", "age": 25, "skills": ["Python", "SQL"]}'
data = json.loads(json_text)

print(data)
print(type(data))
print(data["name"])

Output:

{'name': 'Alice', 'age': 25, 'skills': ['Python', 'SQL']}
<class 'dict'>
Alice

6. Read JSON from a file

Use json.load() to read JSON from a file.

Example file data.json:

{
  "name": "Bob",
  "age": 30,
  "city": "Vancouver"
}

Python code:

import json

with open("data.json", "r", encoding="utf-8") as f:
    data = json.load(f)

print(data)
print(data["city"])

7. Write JSON to a file

Use json.dump() to save a Python object as JSON in a file.

import json

data = {
    "name": "Bob",
    "age": 30,
    "city": "Vancouver"
}

with open("output.json", "w", encoding="utf-8") as f:
    json.dump(data, f, indent=2)

10. Access nested JSON data

import json

json_text = '''
{
  "company": "OpenAI",
  "employees": [
    {"name": "Alice", "role": "Engineer"},
    {"name": "Bob", "role": "Designer"}
  ]
}
'''

data = json.loads(json_text)

print(data["company"])
print(data["employees"][0]["name"])
print(data["employees"][1]["role"])

Output:

OpenAI
Alice
Designer

11. Common errors

Invalid JSON often causes json.JSONDecodeError.

import json

bad_json = "{name: 'Alice'}"
data = json.loads(bad_json)

This fails because:

  • The key is not in double quotes.
  • The string uses single quotes.

Safer version:

import json

json_text = '{"name": "Alice", "age": 25}'

try:
    data = json.loads(json_text)
    print("Valid JSON:", data)
except json.JSONDecodeError as e:
    print("Invalid JSON")
    print(e)

12. Useful options

Use ensure_ascii=False

This is helpful when your data contains Chinese or other non-English characters.

import json

data = {
    "name": "张三",
    "city": "温哥华"
}

print(json.dumps(data, ensure_ascii=False, indent=2))

Use sort_keys=True

This sorts dictionary keys alphabetically in the output.

import json

data = {"b": 2, "a": 1, "c": 3}
print(json.dumps(data, indent=2, sort_keys=True))

13. Full practical example

import json

student = {
    "name": "Peter",
    "age": 16,
    "subjects": ["Math", "Science", "English"],
    "graduated": False
}

# Python object -> JSON string
json_text = json.dumps(student, indent=2)
print("JSON text:")
print(json_text)

# Save to file
with open("student.json", "w", encoding="utf-8") as f:
    json.dump(student, f, indent=2, ensure_ascii=False)

# Read back from file
with open("student.json", "r", encoding="utf-8") as f:
    loaded_student = json.load(f)

print("\nLoaded from file:")
print(loaded_student)
print("Student name:", loaded_student["name"])
