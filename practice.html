Goal: move left to right, increase the step number, and wrap at the edge

            

            
Medium

            Move from left to right, writing the step number into the current cell. Increase the number by 1 each move.

            Print the grid after every move until the whole board is filled.
          


          

          

            
            
            
          


          

            Sample solution
            
N = 5
grid = [[0] * N for _ in range(N)]

def print_grid(board):
    for row in board:
        print(" ".join(str(cell) for cell in row))
    print()

col = 0

for step in range(1, N * N + 1):
    grid[row][col] = step
    print_grid(grid)

    col += 1
    if col == N:
        col = 0
        row += 1

5.13 Bonus. Knight tour on a 5x5 grid

              
Goal: fill a 5x5 grid step by step and print after each move

            

            
Challenge

          

          

            Create a 5x5 grid filled with 0. Start a knight at grid[0][0].
            Visit all grid cells, increase the current cell number by one each step, and print the grid after every move.
            One possible solution is enough.
            

              Need help reading the sample code?
              Open the knight-tour explanation page.
            

          


          

          

            
            
            
          


          

            Sample solution
            
N = 5
moves = [
    (2, 1), (1, 2), (-1, 2), (-2, 1),
    (-2, -1), (-1, -2), (1, -2), (2, -1)
]

def print_grid(board):
    for row in board:
        print(" ".join(str(cell) for cell in row))
    print()

def is_valid(r, c, board):
    return 0 <= r < N and 0 <= c < N and board[r][c] == 0

def onward_count(r, c, board):
    count = 0
    for dr, dc in moves:
        nr, nc = r + dr, c + dc
        if is_valid(nr, nc, board):
            count += 1
    return count

def knight_tour(r, c, step, board, path):
    board[r][c] = step
    path.append((r, c))

    if step == N * N:
        return True

    next_moves = []
    for dr, dc in moves:
        nr, nc = r + dr, c + dc
        if is_valid(nr, nc, board):
            next_moves.append((onward_count(nr, nc, board), nr, nc))

    next_moves.sort()

    for _, nr, nc in next_moves:
        if knight_tour(nr, nc, step + 1, board, path):
            return True

    board[r][c] = 0
    path.pop()
    return False

search_grid = [[0] * N for _ in range(N)]
path = []
knight_tour(0, 0, 1, search_grid, path)

grid = [[0] * N for _ in range(N)]
step = 1
for r, c in path:
    grid[r][c] = step
    print_grid(grid)
    step += 1

Q27. Inheritance with speak()

              
Lesson match: OOP - Class Inheritance

            

            
Hard
