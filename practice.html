1) Variables & print

Q1. Hello with a variable
Goal: store a name and print a sentence
Easy
Make a variable called name and store your name. Print:
Hello, my name is ____
Sample solution 
name = "Avi"
print(f"Hello, my name is {name}")
Q2. Add two numbers
Goal: variables + math + print
Easy
Create a = 5 and b = 8. Print their sum.
Sample solution 
a = 5
b = 8
print(a + b)
Q3. Update a variable
Goal: change a variable value
Easy
Create a variable score with any number, print it, then change it and print again.
Sample solution 
score = 10
print(score)
score = 15
print(score)

2) Strings & formatting

Q4. Favorite animal
Goal: string variable + print
Easy
Create animal = "dog". Print: My favorite animal is dog
Sample solution 
animal = "dog"
print("My favorite animal is " + animal)
Q5. f-string age sentence
Goal: use an f-string
Easy
Use an f-string to print: Tom is 12 years old (You can store name and age first.)
Sample solution 
name = "Tom"
age = 12
print(f"{name} is {age} years old")
Q6. Join strings
Goal: combine two strings
Easy
Combine "Hello" and "World" into one line: Hello World
Sample solution

3) If statements

Q7. Teenager check
Goal: if statement
Medium
Given age = 15, if age is greater than 12, print Teenager.
Sample solution 
age = 15
if age > 12:
    print("Teenager")
Q8. Even or odd
Goal: input + modulo
Medium
Ask the user for a number. Print Even if it is divisible by 2, else print Odd.
Sample solution 
n = int(input("Number: "))
if n % 2 == 0:
    print("Even")
else:
    print("Odd")

4) Loops

Q9. Count 1 to 10
Goal: for loop + range
Medium
Print numbers from 1 to 10 using a loop.
Sample solution 
for i in range(1, 11):
    print(i)
Q10. Star pattern
Goal: nested idea / string repeat
Medium
Print: 
*
**
***
****
*****
Use a loop.
Sample solution 
for i in range(1, 6):
    print("*" * i)
Q11. Even numbers 2 to 20
Goal: step in range
Medium
Print even numbers from 2 to 20.
Sample solution 
for n in range(2, 21, 2):
    print(n)

5) Lists & functions

Q12. List + loop
Goal: list + for loop
Medium
Create a list called animals with 3 animals. Print each one using a loop.
Sample solution 
animals = ["dog", "cat", "rabbit"]
for a in animals:
    print(a)
