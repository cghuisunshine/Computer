1. Basics

JSON is a text format for structured data. In Python, JSON is usually converted into normal Python objects like dictionaries and lists.

Python object
data = {
    "name": "Alice",
    "age": 25,
    "is_student": False,
    "skills": ["Python", "SQL"],
    "address": None
}
Equivalent JSON
{
  "name": "Alice",
  "age": 25,
  "is_student": false,
  "skills": ["Python", "SQL"],
  "address": null
}

Key conversions: False → false, True → true, None → null.

Q1. Convert a dictionary to JSON

                
Goal: use json.dumps()

              

              
Easy

            

            

              Create a dictionary called student with keys name and grade.
              Print it as formatted JSON using indent=2.
            

            
            

              
              
              
              
            

            

              
Playground output

              

            

            

              Sample solution
              
import json

student = {"name": "Mia", "grade": 8}
print(json.dumps(student, indent=2))

            

          


          

            

              

                
Q2. Read a value from JSON text

                
Goal: use json.loads()

              

              
Easy

            

            

              Make a variable json_text equal to '{"city": "Vancouver", "temp": 12}'.
              Convert it to Python and print the city.
            

            
            

              
              
              
              
            

            

              
Playground output

              

            

            

              Sample solution
              
import json

json_text = '{"city": "Vancouver", "temp": 12}'
data = json.loads(json_text)
print(data["city"])

            

          


          

            

              

                
Q3. Access nested JSON data

                
Goal: parse then index nested values

              

              
Medium

            

            

              Convert this JSON text into Python, then print the second skill:
              '{"user": {"name": "Ava", "skills": ["Python", "SQL", "Pandas"]}}'
            

            
            

              
              
              
              
            

            

              
Playground output

              

            

            

              Sample solution
              
import json

json_text = '{"user": {"name": "Ava", "skills": ["Python", "SQL", "Pandas"]}}'
data = json.loads(json_text)
print(data["user"]["skills"][1])

            

          


          

            

              

                
Q4. Catch invalid JSON

                
Goal: use json.JSONDecodeError

              

              
Medium

            

            

              Create a broken JSON string like '{"name": "Ava", age: 12}'.
              Use try/except so the program prints Invalid JSON instead of crashing.
            

            
            

              
              
              
              
            

            

              
Playground output

              

            

            

              Sample solution
              
import json

broken = '{"name": "Ava", age: 12}'

try:
    json.loads(broken)
except json.JSONDecodeError:
    print("Invalid JSON")

            

          


          

            

              

                
Q5. Write JSON to a file and read it back

                
Goal: use json.dump() and json.load()

              

              
Medium

            

            

              Create a dictionary, save it to user.json, then read the file back and print the value of name.
            

            
            

              
              
              
              
            

            

              
Playground output

              

            

            

              Sample solution
              
import json

user = {"name": "Noah", "city": "Vancouver"}

with open("user.json", "w", encoding="utf-8") as f:
    json.dump(user, f, indent=2)

with open("user.json", "r", encoding="utf-8") as f:
    loaded = json.load(f)

print(loaded["name"])
