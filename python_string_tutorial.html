1. What is a string?

A string is text in Python.

name = "Peter"
message = "Hello, world!"
print(name)
print(message)

2. Create strings

word = "apple"
sentence = "I like Python."
empty = ""

Strings are written inside quotation marks.

3. Single quotes and double quotes

a = 'hello'
b = "world"

print(a)
print(b)

Both single quotes and double quotes work.

4. Multi-line strings

text = """This is line 1.
This is line 2.
This is line 3."""

print(text)

5. Access characters by index

Strings use index positions like lists. The first character is at index 0.

word = "python"

print(word[0])   # p
print(word[1])   # y
print(word[-1])  # n
Strings are immutable. That means you can read characters, but you cannot directly change one character inside a string.

6. Loop through a string

word = "cat"

for ch in word:
    print(ch)

7. Length of a string

Use len():

text = "banana"
print(len(text))
6

8. String slicing

word = "python"

print(word[0:2])   # py
print(word[2:5])   # tho
print(word[:4])    # pyth
print(word[3:])    # hon

9. Combine strings

Use + to join strings.

first = "Hello"
second = "World"
result = first + " " + second

print(result)
Hello World

10. Repeat strings

print("ha" * 3)
hahaha

11. Check text inside a string

text = "I like Python"

print("Python" in text)   # True
print("Java" in text)     # False

12. Common string methods

text = "  hello world  "

print(text.upper())
print(text.lower())
print(text.strip())
print(text.title())
print(text.capitalize())

Useful methods:

  • upper() makes all letters uppercase
  • lower() makes all letters lowercase
  • strip() removes spaces at the beginning and end
  • title() makes each word start with a capital letter
  • capitalize() makes only the first letter uppercase

13. Replace text

text = "I like cats"
new_text = text.replace("cats", "dogs")

print(new_text)
I like dogs

14. Split and join

split()

text = "red,green,blue"
colors = text.split(",")
print(colors)
['red', 'green', 'blue']

join()

words = ["I", "love", "Python"]
sentence = " ".join(words)
print(sentence)
I love Python

15. Format strings

Using f-strings:

name = "Amy"
age = 12

print(f"My name is {name} and I am {age} years old.")

Using format():

name = "Amy"
age = 12

print("My name is {} and I am {} years old.".format(name, age))

16. Escape characters

print("He said, \"Hello!\"")
print("Line1\nLine2")
print("A\tB")

Common escape characters:

  • \" double quote
  • \n new line
  • \t tab

18. Summary

A Python string lets you:

  • store text
  • access characters by index
  • slice part of the text
  • combine and repeat text
  • use methods like upper(), lower(), and replace()
  • format text with f-strings
Strings are one of the most important Python data types. You will use them in almost every program.
