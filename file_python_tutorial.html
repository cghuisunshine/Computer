1. What file operations are

File operations let Python programs read data from files and write data back to files. This is useful for saving notes, loading text, storing results, or keeping simple records.

The basic tool is the built-in open() function. 

f = open("test.txt", "r")
In most beginner programs, you work with text files first. Python can also work with binary files such as images by using binary modes like "rb" and "wb".

2. Open a file

This example opens a file, reads all of its content, prints it, and then closes the file manually.

f = open("test.txt", "r")
content = f.read()
print(content)
f.close()

3. Better way: use with

Using with is safer because Python closes the file automatically when the block finishes.

with open("test.txt", "r") as f:
    content = f.read()
    print(content)
Use with open(...) whenever possible. It is the standard pattern in real Python code.

4. Write to a file

Mode "w" writes a file from the beginning and overwrites old content.

with open("test.txt", "w") as f:
    f.write("Hello\n")
    f.write("Python\n")

After this, test.txt will contain:

Hello
Python

5. Append to a file

Mode "a" adds content to the end of an existing file without deleting the old content.

with open("test.txt", "a") as f:
    f.write("More text\n")

6. Read the whole file

Use read() when you want the full file as one string.

with open("test.txt", "r") as f:
    data = f.read()
    print(data)

7. Read line by line

Looping through the file is useful when the file is large or when you want to process one line at a time.

with open("test.txt", "r") as f:
    for line in f:
        print(line.strip())

strip() removes the ending \n.

8. Read all lines into a list

Use readlines() when you want a list where each item is one line from the file.

with open("test.txt", "r") as f:
    lines = f.readlines()

print(lines)

Example result:

['Hello\n', 'Python\n', 'More text\n']

9. Write a list of lines

You can write many lines at once with writelines().

lines = ["apple\n", "banana\n", "orange\n"]

with open("fruits.txt", "w") as f:
    f.writelines(lines)

10. Common file modes

Reading and writing

  • "r": read
  • "w": write and overwrite
  • "a": append at the end
  • "x": create a new file

Binary mode

  • "rb": read binary data
  • "wb": write binary data
  • Useful for images, audio, and other non-text files

11. File exists check

Before reading or deleting a file, it is often safer to check whether the file exists.

import os

if os.path.exists("test.txt"):
    print("File exists")
else:
    print("File does not exist")

12. Delete and rename a file

Python can also manage files with the os module.

Delete a file:

import os

if os.path.exists("test.txt"):
    os.remove("test.txt")
    print("Deleted")
else:
    print("Not found")

Rename a file:

import os

os.rename("old.txt", "new.txt")

13. Copy one file to another

A simple way to copy a text file is to read from one file and write the same content into another.

with open("source.txt", "r") as f1:
    data = f1.read()

with open("target.txt", "w") as f2:
    f2.write(data)

14. UTF-8 text files

For Chinese or other non-English text, it is safer to specify encoding="utf-8".

with open("notes.txt", "r", encoding="utf-8") as f:
    print(f.read())

Write example:

with open("notes.txt", "w", encoding="utf-8") as f:
    f.write("你好\nHello\n")

15. Full small example

filename = "demo.txt"

# write
with open(filename, "w", encoding="utf-8") as f:
    f.write("Line 1\n")
    f.write("Line 2\n")

# read
with open(filename, "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())

Output:

Line 1
Line 2

16. Common mistake

This code will fail if the file does not exist:

with open("abc.txt", "r") as f:
    print(f.read())

Safer version:

import os

filename = "abc.txt"

if os.path.exists(filename):
    with open(filename, "r", encoding="utf-8") as f:
        print(f.read())
else:
    print("File not found")
Reading with mode "r" requires the file to already exist. If it does not, Python raises an error.

17. For Thonny

If you run Python in Thonny, the file is usually created in the current working folder. You can check it with:

import os
print(os.getcwd())

That shows where your file is being read or written.

18. Beginner practice example

name = input("Enter your name: ")

with open("names.txt", "a", encoding="utf-8") as f:
    f.write(name + "\n")

with open("names.txt", "r", encoding="utf-8") as f:
    print("\nAll names:")
    print(f.read())
This is a good beginner exercise because it uses input, append mode, and reading from a file in one small program.

19. Summary

  • Use with open(...) whenever possible.
  • Use "r" to read, "w" to overwrite, and "a" to append.
  • Use encoding="utf-8" for text with Chinese or other Unicode characters.
  • Use os.path.exists() before reading or deleting when the file may not exist.
